# 💝 FlirtGPT

A web application that analyzes social media profiles to help you make better connections. Built for AI Cup 2025.

## 🚀 Features

- Search for social media profiles by name
- Analyze profiles from LinkedIn, Facebook, and Instagram
- Get personality insights and interests
- Beautiful, minimalistic UI built with Streamlit

## 🛠️ Setup

1. Clone the repository:
```bash
git clone <repository-url>
cd flirtgpt
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
streamlit run app.py
```

## 🌐 Deployment

The application is ready to be deployed on:
- Vercel
- Render
- Hugging Face Spaces

## 📝 Usage

1. Enter a full name in the input field
2. Click "Analyze Profile"
3. View the found profiles and personality analysis
4. Click on profile links to visit the social media pages

## ⚠️ Limitations

- Uses Google search to find profiles (no API key required)
- Results may vary based on profile visibility
- Personality analysis is generated using predefined templates
